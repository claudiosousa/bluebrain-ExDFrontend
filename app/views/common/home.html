Get started with the Neurorobotics platform

The Neurorobotics Platform is an Internet-accessible simulation system that allows the simulation of robots controlled by spiking neural networks. For general information visit neurorobotics.net.

This page guides you through the menu items, telling you how to play around with template experiments, create a Collab and populate it with custom experiments.

Video tutorials

Watching a video tutorial is the fastest way to get started. Our tutorial covers all basic steps to customize your experiments.

Example experiments

This menu item takes you to the list of our template experiments. With the basic account you can join experiments run by others.

Request a Test or Project account to create and edit your own experiments, interact with the environment and download configurations to your computer.

Create your Collab

In order to set up your own experiments, create a new Collab you can fully edit. It is then possible to share data and experiments with invited team members. Experiments become customizable and changes become persistent.

Upgrade your account

In order to run experiments by yourself, upgrade your account to a Test or Project account. Such accounts are limited.

Fill the form in Upgrade your account menu to request them.

Documentation and Support

Documentation provides you with an extensive user manual, for an in depth reference to user interface or editing features.

Support pro gives you all the ways to contact us and get help from the community.
